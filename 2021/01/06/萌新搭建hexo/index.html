萌新搭建hexo

前言

萌新0基础，就是为了学下搭建hexo，太难了，找一堆教程，才把hexo搭建搞明白。
昨天加今天十多个小时可算搭建成功了。
在Github设置的时候就看的很迷糊。

1.搭建hexo教程开始喽！

  1. 安装Git
  2. 安装nodejs
    安装完成后，桌面右键 Git Bash 输入命令，这是检查nodejs安装成不成功。
  3. 安装hexo
    在C盘 D盘 F盘 哪个盘都可以看你自己的选择，创建一个文件夹比如‘hexo’，因为为了安装hexo，在hexo文件夹 右键点击Git bash打开输入命令 安装完成输入 hexo -v查看下版本
    接下来就是真正安装hexo了 还是在hexo文件夹 右键点击Git bash输入命令 在输入命令 新建完成后，指定文件夹目录下有：
  • node_modules: 依赖包
  • public：存放生成的页面
  • scaffolds：生成文章的一些模板
  • source：用来存放你的文章
  • themes：主题
  • ** _config.yml: 博客的配置文件**
  1. 输入命令 在本地打开测试 输入完 就出现 http://locahost:4000 打开可以看到你生成的博客了

2.部署到Github个人创库

首先，你要有一个Github账号
注册成功后登录，在Github.com 左边看到一个New 绿色按钮 这就是创建仓库
创建一个和你用户名相同的仓库,比如我的Github用户名是ibaole，那么我创建域名就是ibaole.github.io。
Repository name 这里填 用户名.github.io 填完 点击create repository

3.生成SSH添加到GitHub

回到Git bash 输入命令 这里用户名填 你的Github用户名，邮箱填 你的Github注册的邮箱

然后创建SSH，按回车就行

这个时候生成了.ssh的文件夹。在你的电脑中找到这个文件夹。
SSH，就是秘钥 id_rsa是你这台电脑的私人秘钥 id_rsa.pub是公共秘钥 打开id_rsa.pub这个文件 看见里面的秘钥全部复制下来
而后在GitHub的setting，找到SSH and GPG keys的设置选项，点击New SSH key，title随便填 key 就是秘钥了粘贴进去 点击Add SSH key 就行了
回到 Git bash 输入命令

出现 Hi 就是成功了

这里你就要创建个index.html 因为是把分支main搞出来
怎么创建呢 看Set up in Desktop 下面有个creating a new file 点击他
